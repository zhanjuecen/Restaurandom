## Summary

This App suggests a random restaurant based on the device's current location
and the user's filtering criteria (walking vs. driving, price range, etc.).
This Project is a prototype made for VandyHacks 2017.

## Technology

##### Angular 4
##### Ionic 3
##### Cordova.
